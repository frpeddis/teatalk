🔑 Connect an AI provider

Paste your Anthropic API key to generate recaps and smart questions with Claude. It is stored only in this browser and sent only to Anthropic.
Keep it private: a key stored in a browser can be read by anyone using this device. Use a key limited to this purpose and remove it when finished. Get a key at console.anthropic.com.
Ollama runs locally on your machine — no API key needed.
Run Ollama locally
Ollama is a free, open-source model server. Start it once before using this provider.
1) Install Ollama from ollama.com (macOS / Windows / Linux). On Linux you can also run:
curl -fsSL https://ollama.com/install.sh | sh
2) Pull the Gemma model (first time only — downloads a few GB):
ollama pull gemma3
3) Start the Ollama server (keeps running in the background):
ollama serve
4) Set endpoint and model below, then test the connection.
To shut it down later:
On Windows / macOS: click the Ollama tray icon → Quit Ollama. On Linux: stop the service or kill the process.
Linux stop: pkill ollama  or  sudo systemctl stop ollama

🧭 Conversation context

Describe the goal of the conversation and attach documents to discuss. This is added to the start of the prompt to guide recaps and questions.
PDF, Word (.docx) and PowerPoint (.pptx) files are parsed in the browser and their text is added to the prompt — together with .txt/.md/.csv/.json/.html. Content is available for recaps and questions as soon as you save, even before transcribing.
